3. Khusus Pembuatan Pertama#

sebelum itu buat dulu repositori github nya buat menyimpan data, dan git clone dengan cara copy link github nya serta masukkan private key nya seperti contoh di bawah.

# !git clone https://{Private Key}@github.com/arosyihuddin/ppw.git

setelah di clone selanjutnya buat jupyterbook nya dengan menggunakan code berikut

# !jupyter-book create ppw1/

setelah itu pindah semua file maupun folder dari ppw1 tadi ke folder github yang telah di clone dengan menggunakan code berikut

# !mv /content/drive/MyDrive/Kuliah\ /Semester\ 7/Pencarian\ Dan\ Penambangan\ Web/ppw1/* /content/drive/MyDrive/Kuliah\ /Semester\ 7/Pencarian\ Dan\ Penambangan\ Web/ppw

# !rm -rf ppw1/
